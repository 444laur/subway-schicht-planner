- backend: python 
- backend framework: flask
- core solver: CP-SAT solver
- database: MySQL
- frontend: HTML, CSS and Javascript

Für jeden Mitarbeiter werden folgende Felder in Ihrer Datenbank benötigt:

Name
Verfügbarkeitsregeln: Wochentag + Startzeit + Endzeit (z. B. „Montag 10:00–18:00 Uhr“). Jede Person kann mehrere Einträge haben, einen pro verfügbarem Tag.
Maximale Wochenstunden
Alter (zur Prüfung, ob das Jugendarbeitsschutzgesetz gilt; für unter 18-Jährige gelten andere gesetzliche Regelungen zu Schichtlänge und Überstunden)
Optional: Wunschfrei (konkrete freie Tage)

Jeder Schicht-/Schichttyp benötigt:

Name (Frühschicht, Spätschicht)
Zeitbereich
Mindestanzahl an Personen erforderlich
Maximal zulässige Personenzahl (optional)
